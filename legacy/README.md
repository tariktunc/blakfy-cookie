# Legacy v1.x

Bu klasör v1.x sürümünün kaynağıdır. v2.0.0'da `src/` ağacına refactor edildi. Geri dönüş güvenliği için saklanır.

- `cookie.js` — v1.2.0 IIFE bundle (47 KB ham)
- `cookie-defaults.js` — v1.0.0 GCM bootstrap
- `locales/` — kullanılmayan i18n JSON dosyaları (5 dil)
- `status.json.v1` — eski statü dosyası şeması

v2 release sonrası kullanıcılar `@1` etiketinde kalmaya devam edebilir; bu klasör buradan değil GitHub'ın `v1` branch'inden servis edilir (v1 son commit'i `v1.2.0` tag'inde).
